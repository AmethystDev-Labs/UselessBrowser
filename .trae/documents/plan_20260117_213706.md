# Frosted Glass UI 重构计划

## 项目分析
- **框架**: PyQt6 + qfluentwidgets
- **当前风格**: 基础 Fluent Design
- **目标风格**: 毛玻璃效果 (Frosted Glass / Glassmorphism)

## 核心改动
1. **主窗口背景**: 添加渐变背景图，实现玻璃透射效果
2. **毛玻璃特效**: 使用半透明背景 + 模糊滤镜
3. **卡片样式**: 圆角 + 半透明背景 + 边框高光
4. **导航栏**: 玻璃质感侧边栏
5. **滚动区域**: 毛玻璃覆盖效果

## 具体实施步骤

### 1. 创建毛玻璃样式表
- 新建 `app/ui_glass_styles.py`
- 定义深色/浅色主题的毛玻璃QSS样式
- 包含：窗口背景、卡片、按钮、输入框等组件样式

### 2. 修改主窗口 (main_window.py)
- 设置窗口背景图片（渐变/风景图）
- 应用全局毛玻璃样式
- 启用窗口透明效果

### 3. 重构UI构建器 (ui_builders.py)
- 为所有页面容器添加玻璃效果
- 修改卡片背景为半透明
- 添加圆角和边框样式
- 调整滚动区域视觉效果

### 4. 添加模糊特效类
- 创建 `GlassEffect` 工具类
- 为重要组件添加 `QGraphicsBlurEffect`
- 支持动态调整模糊强度

### 5. 优化各页面
- **Home页面**: 玻璃卡片容器
- **Launch页面**: 半透明表单卡片
- **Profiles页面**: 玻璃分隔面板
- **Settings页面**: 玻璃设置卡片
- **Onboarding页面**: 玻璃流程页面

## 预期效果
- 现代毛玻璃美学
- 窗口背景模糊透射
- 半透明磨砂玻璃卡片
- 柔和阴影和边框高光
- 流畅的视觉效果
